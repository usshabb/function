Canvas Tab Extension

A Notion-like canvas for your new tab page

Installation Instructions

  1. Open Chrome and navigate to chrome://extensions/
  2. Enable Developer mode (toggle in the top right corner)
  3. Click Load unpacked
  4. Select the folder containing this extension (the folder with manifest.json)
  5. The extension is now installed! Open a new tab to see your canvas
Note: To get the extension folder on your computer, download all the files from this Replit project. You can use the "Download as zip" option or clone the repository.
